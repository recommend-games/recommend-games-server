About Recommend.Games

Recommend.Games strives to recommend the best board games for you. We take the user ratings from BoardGameGeek , apply some black magic, and present recommendations that suit you.

BoardGameGeek (BGG) users can rate each game with a score between 1 (awful – defies game description) and 10 (outstanding – will always enjoy playing). These ratings are then used to rank games on the site. But you cannot just use the average score for the ranking, else games with few ratings would shoot to the top, even if a minimum number of votes is required. (At the moment, the game with the highest average rating and at least 100 votes is {{ topAvg.name }}.) Instead, BGG uses a weighted average by adding a number of "dummy" votes to each game. The effect is that a game's rating is drawn towards the overall average rating across all games on BGG, but as more and more users vote, this "Geek Rating" will get closer to the actual average. Currently, the highest ranked game according to BoardGameGeek is {{ topBayes.name }}. Read more about the ratings on the BGG Wiki .

We don't believe this yields a very good overview over the best games today.

Obviously, this is a matter of opinion, but it's hard to deny that many games from BGG's Top 100 are rather special interest products. No doubt, they are perfectly executed if you are a fan of a particular theme, genre, or mechanic, but it's unlikely a random person would really enjoy games like Twilight Struggle or Star Wars: Rebellion.

Recommend.Games takes a different approach. Instead of just calculating averages, we are trying to find good recommendations for users based on their BoardGameGeek ratings. Recommender engines using collaborative filtering have been a hot topic for some time and Turi Create provides an excellent open source implementation.

The main objective of a recommender system is to predict what items a user might like given their preference for other items. We can absolutely do that – see for example recommended games for Richard Garfield. But we can also find recommendations for user we don't know anything about at all – recommendations for the random person on the street. The result is the default ranking on the main page.

Call us biased – but we believe this is a much better selection for anyone to (seriously) get into boardgaming today.
